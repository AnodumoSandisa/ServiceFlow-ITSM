ServiceFlow-ITSM
Project Overview
This project utilizes a Python based decision engine to map raw user input to formal ITIL v4 categories (Incident, Problem, Change, or Service Request).

Key Features

ITIL Mapping: Automatically categorizes requests based on organisational impact.

Security Integration: Leverages my Cybersecurity postgraduate background to flag potential phishing or security breaches for immediate triage.

Governance-Ready: Provides clear "Consulting Advice" for each ticket, making it easier for non-technical managers to understand the next steps.
